Memorial & Tribute Management System (Django)

1. Developed a full-stack web application using Python (Django) to manage memorial entries and tribute messages.
2. Implemented features to add, update, delete, and view memorial records including Name, DOB, message, and image upload.
3. Built a tribute system allowing users to post and view condolence messages dynamically.
4. Designed and managed an admin panel with complete CRUD operations for efficient data handling.
5. Integrated search and filter functionality to quickly find specific memorial entries.
6. Used SQLite database for data storage and Django ORM for backend operations.
7. Ensured responsive UI and clean user experience using HTML, CSS, and Bootstrap.
8. Version controlled the project using Git & GitHub.
